# 本地文档文件整理设计

## 目标

本地文档树支持把单个文件拖入其他文件夹，并允许从任意文件夹行直接创建文档。

## 交互

- 文件图标前增加独立拖动手柄；点击文件仍按原行为打开。
- 拖动经过文件夹时高亮整行，松开后移动文件。
- 根目录也是合法放置目标。
- 文件夹名右侧增加 `…` 菜单，提供“在此新建文档”；根目录沿用同一能力。
- 只移动文件，不支持拖动文件夹。
- 目标目录存在同名文件时阻止移动并提示，不覆盖。
- 当前打开的文件移动后继续保持打开，并更新到新路径。

## 实现

前端使用原生拖放事件，由拖动手柄启动操作，文件夹行处理放置状态。移动和新建成功后刷新文件树。

Tauri 后端新增安全的本地文件移动命令：复用现有路径校验，限定源文件和目标目录都位于已选择的根目录内，拒绝同名目标并返回移动后的相对路径。文件夹内新建复用现有创建流程并传入目标目录。

## 验证

- 文件可移动到嵌套文件夹及根目录。
- 同目录放置无副作用；同名目标不会覆盖。
- 移动已打开文件后仍可继续编辑和保存。
- 每个文件夹的菜单能在正确目录创建文档。
- 搜索/平铺视图中的文件同样可通过手柄拖动。
